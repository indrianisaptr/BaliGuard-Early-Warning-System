### NB01 — Load & EDA
**Tujuan:** Setup project, load 14 dataset, eksplorasi awal.

**Yang dilakukan:**
- Setup folder `data/raw`, `data/processed`, `data/final`, `models/`
- Load 14 dataset dengan parameter tepat (`xlrd` untuk `.xls`, `parse_dates` untuk Excel)

- Visualisasi tren wisman 2009–2024 dengan highlight COVID

**Output:** Insight awal, plot `eda_wisman_trend.png`

---

### NB02 — Preprocessing Time Series
**Tujuan:** Bersihkan dan standarisasi semua data time series ke format bulanan YYYY-MM.

**Yang dilakukan:**
- Konversi format wide BPS → long (melt)
- Alignment timeline ke periode bulanan (`pd.Period`)
- Handling missing values: forward fill untuk kurs & TPK, interpolasi untuk inflasi
- Hitung `bali_share_pct` = wisman Bali / total wisman nasional × 100
- Export 6 CSV bersih ke `data/processed/`

**Output:** 6 CSV di `data/processed/`, period coverage 2009-01 → 2024-12

---

### NB03 — Text Preprocessing & Sentiment
**Tujuan:** Analisis sentimen ulasan wisatawan multibahasa menggunakan XLM-RoBERTa.

**Yang dilakukan:**
- Load `merged_all_hotels.xlsx` (29.332 ulasan, EN/ID/ZH)
- Preprocessing: lowercase, remove HTML, normalisasi encoding
- Deteksi bahasa dengan `langdetect`
- Batch inference `cardiffnlp/twitter-xlm-roberta-base-sentiment` (batch size 32)
- Skor -1 (negatif) s.d. 1 (positif) per ulasan
- Agregasi ke bulanan: `avg_sentiment_monthly`, `pct_negative_monthly`

**Output:** `monthly_sentiment.csv`, `all_reviews_sentiment.csv`

## 6. ⚙️ Feature Engineering — 17 Fitur

Model akhir menggunakan **17 fitur** total (13 core + 4 lag/trend), ditentukan di NB05.

### FEATURES_CORE (13 fitur)
```python
FEATURES_CORE = [
    # Kunjungan wisatawan
    'wisman_growth_mom',    # % perubahan MoM
    'wisman_growth_yoy',    # % perubahan YoY
    'wisman_zscore',        # z-score anomali (rolling 36 bulan)
    # Ekonomi
    'usd_idr_avg',          # kurs rata-rata bulanan
    'usd_volatility_3m',    # std dev kurs 3 bulan terakhir
    'usd_change_mom',       # % perubahan kurs MoM
    # Hotel
    'tpk_bintang',          # tingkat hunian hotel bintang (%)
    'tpk_change_mom',       # perubahan TPK MoM (pp)
    # Makro
    'inflasi_processed',    # inflasi Bali bulanan (%)
    'bali_share_pct',       # pangsa Bali dari wisman nasional (%)
    # Sentimen
    'avg_sentiment_monthly',# rata-rata skor sentimen ulasan
    # Temporal
    'month_num',            # nomor bulan (1–12) untuk seasonality
    'is_peak_season',       # binary: Jul-Okt (peak) = 1
]
```

### FEATURES_EXTENDED (+4 fitur lag/trend)
```python
FEATURES_LAG_TREND = [
    'wisman_ma3',           # moving average 3 bulan
    'wisman_trend_3m',      # slope trend wisman 3 bulan (koefisien regresi linear)
    'bali_share_change',    # perubahan bali_share_pct MoM
    'sentiment_trend_3m',   # slope trend sentimen 3 bulan
]
```

### Perbaikan Khusus Nilai Ekstrem (COVID)
```python
# pct_change() menghasilkan inf ketika nilai sebelumnya 0 (COVID recovery)
df['wisman_growth_mom'] = (df['wisman'].pct_change()
                           .replace([np.inf, -np.inf], np.nan)
                           .clip(-1.0, 10.0))
# Clip 1%-99% sebelum masuk model
df_model[FEATURES] = df_model[FEATURES].clip(
    lower=df_model[FEATURES].quantile(0.01),
    upper=df_model[FEATURES].quantile(0.99), axis=1)
```

## 8. 🤖 NB05 — ML Modeling & Explainability

### Dataset Modeling
- **Input:** `master_dataset_clean.parquet` → setelah dropna = **181 baris × 17 fitur**
- **Periode:** 2009-12 → 2024-12

### Model 1 — Isolation Forest (Anomaly Detection)
```python
iso_forest = IsolationForest(
    n_estimators=200,
    contamination=0.15,   # estimasi 15% data adalah anomali
    random_state=42,
    n_jobs=-1
)
```

**Hasil:**
- Anomali terdeteksi: **27 bulan** (14.9%)
- Overlap dengan periode COVID (2020–2021): **17/24 bulan = 70.8%**
- Crosstab: KRISIS → 9 ANOMALI, 4 NORMAL; SIAGA → 11 ANOMALI, 35 NORMAL

### Model 2 — Random Forest Classifier
```python
rf_model = RandomForestClassifier(
```

**Evaluasi dengan Custom Walk-Forward CV (COVID-aware):**
- Strategi: min_training = 108 baris (60% data), test = 18 baris/fold, gap = 1 bulan

### 5 Step Pipeline
```
[1/5] Fetch USD/IDR terbaru
  → Bulan berjalan: open.er-api.com (real-time)
  → Historis: Frankfurter (ECB, reliable)
  → Fallback: cache lokal

[2/5] Cek update data BPS (manual append)

[3/5] Rebuild fitur engineering
  → Fix inf/NaN dari pct_change()
  → Clip 1%-99% untuk outlier COVID

[4/5] Hitung ulang crisis score

[5/5] Jalankan model predictions
  → Auto-detect n_features_in_ dari model
  → Safety net: impute NaN sebelum scaling
  → Fallback rule-based jika model gagal
```

### Penanganan Infinity (Perbaikan Khusus)
```python
# pct_change() dari nilai 0 = inf (COVID recovery)
df['wisman_growth_mom'] = (df['wisman'].pct_change()
                           .replace([np.inf, -np.inf], np.nan)
                           .clip(-1.0, 10.0))

# Safety net sebelum scaler.transform()
X = np.where(np.isinf(X), np.nan, X)
col_medians = np.nanmedian(X, axis=0)
X[nan_mask] = np.take(col_medians, np.where(nan_mask)[1])
```

```python
expected_n = scaler.n_features_in_  # baca dari model
```
